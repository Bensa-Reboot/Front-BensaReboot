## 📌 Sobre o projeto
 
A **Bensa StreetWear** é um e-commerce fictício de moda urbana desenvolvido com base em protótipo de alta fidelidade elaborado no Figma. O site contempla o fluxo completo de compra: navegação por categorias e marcas, página de produto, carrinho, checkout e área do usuário.
